Программа считает отношение суммарного количества решений сравнения P^2=D(N) для N < Nmax к суммарному количеству примитивных решений уравения x^2-Dy^2=N для N<Nmax.

Вычисления можно проводить: 
 1)	для простых D
 2)	для всех D

Можно вычислять для:
1) 1 < N < Nmax
2) -Nmax < N < Nmax

Задаются границы вычислений для D (Dmin, Dmax).
